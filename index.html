What's all this then?

All the music is randomly generated, within certain musical constraints, using the excellent Tone.js. The random generation algorithm is seeded with the seed shown above, this can be changed to anything you like, and entering the same seed will always produce the same randomly generated song.

All the code, and a bit more info, can be found on GitHub here: https://github.com/wheelibin/synaesthesia
